## Getting Started

1. Install the [flutter framework](https://docs.flutter.dev/get-started/install)
2. Clone the repo
   ```sh
   git clone https://github.com/ANicon111/KeyRunner.git
   ```
3. Build the app
   ```sh
   cd /path/to/KeyRunner
   flutter pub get
   flutter build
   ```
